💨Migration to AWS cloud server

Based on an old existing vending program, stop to use its old backend server platform(VMware) and re-developed a new 3rd-party program(Python) that keep monitoring the real-time log records from the old vending program’s log file in the same embedded computer(Linux) and forward those real time records’ data(e.g Transaction records, machine status) to a new backend server platform(AWS cloud) for more than 200 refill water stations & 300 vending machines.
